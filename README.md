# Real Estate Valuation AI üè°

A Data Science and Machine Learning project that predicts house prices and categories, deployed as an interactive Streamlit app with AI-powered insights.

## Features
- **Price Prediction**: Optimized XGBoost Regressor (Log-target transformation).
- **Category Classification**: 'Budget', 'Standard', or 'Luxury' classification using Random Forest.
- **AI Explanations**: Context-aware market insights using Llama-3-70B (via Groq).
- **Interactive UI**: Built with Streamlit.

## Files
- `real_estate_project.ipynb`: Core Notebook (EDA, Modeling, Deployment Code).
- `app.py`: Streamlit application.
- `presentation_slides.md`: Project presentation slides.
- `requirements.txt`: Python dependencies.
- `*.pkl`: Trained models (generated by notebook).
- `data.csv`: Dataset.

## How to Run locally

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Train Models**:
    Run `real_estate_project.ipynb` to generate models (`.pkl`) and `app.py`.

3.  **Run App**:
    ```bash
    streamlit run app.py
    ```

## Cloud Deployment (Streamlit)
1.  Fork to GitHub.
2.  Deploy `app.py` on Streamlit Cloud.
3.  Add `GROQ_API_KEY` to Streamlit Secrets for AI features.
